# MyFitnessPal Screenshot OCR Plan

Status: superseded by the Terra Connected Apps integration.

The proposed screenshot and OCR workflow will not be built. MyFitnessPal data should arrive through Terra once Gordy's production provider access is enabled, avoiding duplicate uploads and manual transcription.

Until Terra is live, clients can still add daily totals manually as a fallback or correction. The assigned nutrition plan remains the primary coaching workflow, and legacy entries named `MyFitnessPal totals` remain supported so no historical nutrition data is lost.

See `docs/terra-production-checklist.md` for the production integration steps.
